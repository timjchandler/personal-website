I'm a masters student in artificial intelligence at the university of Edinburgh.
This website is where I will host AI projects, games and any other things that I work on, or think are interesting.

programming

I enjoy playing around with code, writing programs and learning new languages. Currently I'm to learning Rust, as well as the key web languages. I'm also continuously improving my python, c++ and java.
You can see some of my work on github. I intend to try and work on a small project each month in 2022. Let's see how long this intention lasts!

computer graphics

It's always satisfying to be able to really see the results of what you're working on. Computer graphics is great for that, even when mistakes are made and a previously innocuous model turns into an abomination of colours and triangles.
I do a decent amount of messing about in blender, but particularly enjoy working with shaders, both through node editors and direct programming. There are just so many more possibilities for visual monstrosities!

games

Games are a great opportunity for giving a bit more life to some of the computer graphics projects.
